# Team Game Night Hub

A fun, zero-friction platform for remote teams to host virtual game nights.

## What is this?

Team Game Night Hub is a web app designed to make virtual team bonding easy and fun. Just open it up during your next team call, share your screen, and start playing! Fresh questions are generated by AI every time you visit.

## Features

- **Built-in Games** - Ready-to-play games with no setup required
- **AI-Powered Questions** - Fresh prompts generated automatically using Google Gemini
- **External Game Directory** - Curated links to popular multiplayer web games
- **Works Everywhere** - Optimized for Zoom, Google Meet, and Microsoft Teams
- **No Account Needed** - Just open and play

## Available Games

| Game | Description |
|------|-------------|
| Global Trivia | Test your team's knowledge on history, science, and pop culture |
| Would You Rather | Classic game of impossible choices that sparks debates |
| Icebreakers | Lighthearted questions to help teammates get to know each other |
| This or That | Quick-fire preference questions for rapid discussion |

## How to Use

1. **Start a game** - Click any game card on the home page
2. **Share your screen** - Let everyone see the questions on your video call
3. **Navigate prompts** - Use Next/Previous buttons or shuffle for variety
4. **Get fresh questions** - Click "Get Fresh Questions (AI)" for new content
5. **Reveal answers** - For trivia, click to show the answer after guessing

**Pro tip:** Go full screen for the best presentation experience!

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env.local` file and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Admin Settings

Visit the Admin page to:
- View all built-in games and their prompt counts
- Add, edit, or remove external game links
- All changes are saved to your browser's local storage
